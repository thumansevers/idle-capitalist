# 💰 资本家模拟器 (Idle Capitalist)

一款挂机赚钱游戏，参考《资本家大冒险》核心玩法。纯单机，自己玩！

## ✨ 游戏特色

### 🌍 多星球探索
| 星球 | 解锁条件 | 建筑数量 |
|------|----------|----------|
| 地球 🌍 | 初始解锁 | 6 个 (柠檬水摊→石油公司) |
| 月球 🌙 | 石油公司 Lv.10 | 3 个 (氧气吧→氦 3 矿) |
| 火星 🔴 | 氦 3 矿 Lv.10 | 3 个 (生态穹顶→星际港口) |

### 📈 核心玩法
- **进度条系统** - 每个建筑独立计时，自动赚钱
- **倍率系统** - 重置宇宙获得永久倍率
- **升级商店** - 点击强化、全局增益、星球倍率
- **批量购买** - x1 / x10 / x100 / xMAX 一键切换
- **阶段增强** - 每 10/100/1000 个触发收益/时间奖励
- **离线收益** - 关闭游戏也赚钱 (50% 效率)
- **自动存档** - 30 秒自动保存

### 🎯 倍率机制
```
重置倍率 = 总赚取 / 1,000,000 × 10%
例：赚过 1 亿 → 重置获得 x10 倍率
```

### ⚡ 升级类型
| 类型 | 效果 | 示例 |
|------|------|------|
| 点击强化 | 手动点击赚更多 | +1$, +5$ |
| 全局增益 | 所有收益 +X% | +10%, +25% |
| 星球倍率 | 特定星球 x2/x5 | 地球 x2 |
| 永恒之力 | 重置倍率 +X% | +10% |

### 🔢 批量购买
| 倍率 | 说明 |
|------|------|
| x1 | 单次购买 1 个 |
| x10 | 单次购买 10 个 |
| x100 | 单次购买 100 个 |
| xMAX | 购买当前能负担的最大数量 |

点击底部 🔢 按钮切换倍率。

### 🏆 阶段增强
每个建筑达到一定数量后触发永久增强：

| 数量 | 收益倍率 | 时间缩减 |
|------|----------|----------|
| 10 个 | x1.5 | -20% |
| 100 个 | x2 | -30% |
| 1000 个 | x3 | -50% |

**叠加计算**：1000 个时收益 = 基础 x 等级 x 1.5 x 2 x 3 = 基础 x 等级 x 9
